## Content Creation Tools

| Tool | Purpose |
|------|---------|
| **edge-tts** | Text-to-speech narration (Microsoft voices, free) |
| **vercel** | Deploy lessons to web |
| **pandoc** | Convert lesson content between formats |
| **marp-cli** | Create presentation slides from markdown |
